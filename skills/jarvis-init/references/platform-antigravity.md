# Platform Setup: Antigravity

## Hooks

Antigravity does not support hooks as of March 2026. The agent must manually run `/jarvis-reload` at the start of each session.

## Instruction file

Read the project's `AGENTS.md` (create it if it doesn't exist). If it does not already contain a `## JaRVIS` section, append the contents of `references/AGENTS.md.example` to the end of the file (preceded by a blank line).
